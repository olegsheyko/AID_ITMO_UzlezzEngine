# Практическое занятие №1 — Каркас игрового движка

C++17, OpenGL 3.3, GLFW, GLAD, CMake

## Что реализовано

- Класс Application с главным циклом и deltaTime
- Система логирования (вывод в консоль и файл engine.log)
- Адаптер рендеринга — интерфейс IRenderAdapter и реализация под OpenGL
- Система состояний: LoadingState → MenuState → GameplayState
- Отрисовка треугольника через шейдеры
- Управление объектом с клавиатуры и мыши

## Управление

- Стрелки — движение треугольника
- ЛКМ — увеличить масштаб
- СКМ — уменьшить масштаб
- ПКМ — повернуть на 45°

## Сборка

Нужно установить GLFW и GLAD в папку external/.

```
cmake -B build
cmake --build build --config Release
```

## Зависимости

- GLFW 3.x
- GLAD (OpenGL 3.3 Core)
- CMake 3.16+
- Visual Studio 2022 / компилятор с поддержкой C++17
